# WEEB社交系统开发环境与工具

## 一、开发环境

### 1.1 基础开发环境
- **操作系统**: Windows 10/11 (主要开发环境)
- **Java开发环境**:
  - JDK 17 (Oracle/OpenJDK)
  - Maven 3.6+ (项目构建工具)
- **Node.js环境**:
  - Node.js 16+ (前端构建支持)
  - npm 8+ (包管理器)

### 1.2 集成开发环境(IDE)
- **后端开发**: IntelliJ IDEA 2023+
- **数据库管理**:
  - MySQL Workbench 8.0+
  - DBeaver (通用数据库工具)
- **API测试**: Postman 或 Apifox
- **版本控制**: Git + GitHub/GitLab

## 二、编程语言与框架版本

### 2.1 后端技术栈
- **核心语言**: Java 17
- **主框架**: Spring Boot 3.5.4
- **安全框架**: Spring Security 6.x
- **ORM框架**: MyBatis-Plus 3.5.8
- **构建工具**: Maven 3.6+

### 2.2 前端技术栈
- **UI框架**: Element Plus 2.11.5
- **CSS预处理器**: Less 4.2.2
- **构建工具**: Webpack/Vite (通过Less Loader 12.2.0)

### 2.3 开发工具库
- **JSON处理**: Jackson (Spring Boot内置)
- **加密工具**: JJWT 0.11.5
- **网络通信**: OkHttp 4.12.0
- **网络框架**: Netty 4.1.108.Final
- **工具库**: Hutool 5.8.18
- **代码简化**: Lombok 1.18.38
- **缓存框架**: Caffeine (Spring Boot内置)
- **SSH支持**: Apache SSHD 2.14.0
- **反射工具**: Reflections 0.10.2

## 三、数据库与存储

### 3.1 关系型数据库
- **MySQL**: 8.0+
  - 字符集: utf8mb4
  - 排序规则: utf8mb4_unicode_ci
  - 连接池: HikariCP (Spring Boot内置)

### 3.2 缓存系统
- **Redis**: 7.0+
  - 数据缓存
  - 会话存储
  - 分布式锁

### 3.3 搜索引擎
- **Elasticsearch**: 8.18.6
  - 全文搜索
  - 中文分词 (IK Analyzer)
  - 索引管理

### 3.4 文件存储
- **本地存储**: 开发环境
- **云存储**: 阿里云OSS 3.15.1 (生产环境)
  - 图片存储
  - 文件上传
  - CDN加速

## 四、运行支撑环境

### 4.1 应用服务器
- **内嵌服务器**: Tomcat (Spring Boot内置)
- **服务端口**: 8080 (开发环境)
- **WebSocket支持**: Spring WebSocket
- **异步处理**: Spring Async

### 4.2 监控与管理
- **健康检查**: Spring Boot Actuator
- **API文档**: Springdoc OpenAPI 2.6.0
- **日志管理**: Logback (Spring Boot内置)
- **性能监控**: Micrometer (Spring Boot内置)

### 4.3 安全与认证
- **认证方式**: JWT (JSON Web Token)
- **加密算法**: BCrypt (密码加密)
- **权限控制**: Spring Security
- **敏感词过滤**: sensitive-word 0.24.0

## 五、AI与智能服务

### 5.1 AI框架
- **Spring AI**: 1.0.0-M1
- **OpenAI集成**: GPT API支持
- **智能功能**:
  - 文本生成与处理
  - 情感分析
  - 内容摘要
  - 多语言翻译
